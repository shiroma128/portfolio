Accessory Design
概要
30代以上の女性をターゲットにした、架空のアクセサリー販売サイトを制作しました。
落ち着きのある配色と余白を意識し、大人の女性に合う上品で見やすいデザインに仕上げています。
工夫した点
商品一覧は見た目が単調にならないよう、配置や見せ方に変化をつけて、最後まで自然に閲覧できるよう工夫しました。
また、テーマ設定から商品画像・ファビコンの制作、商品紹介文の作成、コーディングまで一貫し、世界観に統一感が出るよう意識しています。
制作範囲
企画 / テーマ設定 / 商品画像・ファビコンの制作 / 商品紹介文の作成 / コーディング
使用言語
HTML / CSS / JavaScript
使用ツール
Visual Studio Code / Canva
